# Hi, I'm Akbar Khan 👋

**Full-stack developer & train-simulation enthusiast** — I build web apps by day and bring railways to life in code by night. 🚂

I'm the collaborater of **BGPro (Broad Gauge Productions)** — a store and community for hyper-realistic Indian broad-gauge train simulation content — and I contribute graphics and tooling improvements to the open-source train-sim ecosystem (Open Rails, TSRE5).

## 🔭 What I've been building

- **🚉 BGPro — Broad Gauge Productions** — Full-stack e-commerce platform for train-sim add-ons and handcrafted TrainBoards: React + TypeScript SPA, Hono API on Vercel serverless, MongoDB Atlas, Cloudflare R2 storage, JWT auth with admin/customer roles, community reviews, and an admin dashboard with sales analytics.
- **🎨 Open Rails graphics modernization** — Upgrading the rendering pipeline of the open-source Open Rails train simulator for modern, realistic visuals.
- **🛠 TSRE5 route-editor enhancements** — Added glTF model support and a full 3D transform gizmo (X/Y/Z move handles) to the C++/Qt route editor used by the MSTS/Open Rails community.
- **✅ QA & testing** — Writing structured test cases and automated tests for web applications.
